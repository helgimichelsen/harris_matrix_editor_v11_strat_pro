Harris Matrix Editor V11 STRAT PRO

Ny STRAT layoutmotor:
- Bygger en ren Harris-DAG: yngre/over -> ældre/under.
- Niveau = 1 + maksimum af alle yngre/overliggende relationer.
- Parallelle lag uden relation placeres på samme stratigrafiske niveau.
- Grene deles og samles som i Harris-eksemplerne A/B/C.
- Topsoil/top låses øverst.
- Unexcavated låses næstnederst.
- Natural/Geology låses nederst.
- Faser og struktur-bokse bevares som grafiske annotations, ikke stratigrafiske relationer.
- Ctrl+klik for multi-select.
- Slet valgte kan slette flere contexts ad gangen.
- Fortryd-knap.
- Fladen ryddes før ny HMCX/JSON åbnes.
- HMCX/JSON/PDF/PNG/SVG/Graph export.
